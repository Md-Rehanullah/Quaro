Contribution Guidelines

Code Quality

Write clean, well-documented code that follows our style guidelines and best practices.

Respectful Communication

Maintain respectful and constructive communication in all interactions with the community.

Testing

Test your changes thoroughly and include appropriate tests when submitting new features.

Documentation

Update documentation when making changes and ensure your code is properly commented.
